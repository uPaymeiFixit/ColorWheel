Color Wheel

Launch project
Color wheel is a script I used to teach my younger brother to code (thus the odd code). It is a transparent circle that rotates around the center of the screen, changing the hue as it goes. This is not a terribly complicated script, but I thought the outcome looked interesting. The script is also configured to display text, one word at a time in the center of the circle. The setup.js file contains functions to make canvas drawing simpler (for my brother).
